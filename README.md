## Desenvolvimento

Instale: `NodeJs` e `Yarn`;
Depois: rode `yarn install` na pasta raiz e `yarn start` para iniciar o aplicativo.

## Tecnologias utilizadas

- Electron;
- React;
- Typescript;
- Chakra UI.
